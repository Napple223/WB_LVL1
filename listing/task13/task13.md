Что выведет данная программа и почему?

```go

func someAction(v []int8, b int8) {
  v[0] = 100
  v = append(v, b)
}

func main() {
  var a = []int8{1, 2, 3, 4, 5}
  someAction(a, 6)
  fmt.Println(a)
}

```

Ответ:

```
Вывод: [100 2 3 4 5].

Слайс является ссылочным типом данных и в функцию
someAction передается указатель на слайс "a", в котором
мы меняем элемент с индексом 0 на 100.
При выполнении append мы переприсваиваем переменной "v" новый слайс,
а не меняем изначальный. Поэтому слайс "a" не изменил длины.
```
